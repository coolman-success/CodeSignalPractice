# String Definition

Which of the given string definitions are **incorrect**?

1. `s = 'abacaba'`
2. `s = "abacaba"`
3. `s = ''abacaba''`
4. `s = ""abacaba""`
5. `s = '''abacaba'''`
6. `s = """abacaba"""`

## Question

- [ ] Definitions 5 and 6.
- [ ] All options are correct.
- [ ] Definitions 2, 4 and 6.
- [ ] Definitions 1, 3 and 5.
- [ ] Definitions 3, 4, 5 and 6.
- [ ] Definitions 3 and 4.

## Answer

- [ ] Definitions 5 and 6.
- [ ] All options are correct.
- [ ] Definitions 2, 4 and 6.
- [ ] Definitions 1, 3 and 5.
- [ ] Definitions 3, 4, 5 and 6.
- [x] Definitions 3 and 4.
