# Dictionary Keys

Consider the following types of objects:

1. `set`
2. `frozenset`
3. `tuple` of mutable objects
4. `tuple` of immutable objects
5. `list` of mutable objects
6. `list` of immutable objects

Which of them can be used as dictionary keys?

## Question

- [ ] Objects 2, 4 and 6.
- [ ] Objects 2 and 4.
- [ ] Any of these objects.
- [ ] Objects 2, 3 and 4.
- [ ] Objects 1, 2, 4 and 6.
- [ ] Objects 1, 2, 3, and 4.
- [ ] Objects 4, 6.

## Answer

- [ ] Objects 2, 4 and 6.
- [x] Objects 2 and 4.
- [ ] Any of these objects.
- [ ] Objects 2, 3 and 4.
- [ ] Objects 1, 2, 4 and 6.
- [ ] Objects 1, 2, 3, and 4.
- [ ] Objects 4, 6.
