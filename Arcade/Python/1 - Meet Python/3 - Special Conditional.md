# Special Conditional

Given two boolean variables `a` and `b`, one of the following statements works differently than the others. Which one is it?

## Questions

- [ ] a == not b
- [ ] not (a == b)
- [ ] a == (not b)
- [ ] not a == b

## Answers

- [x] a == not b
- [ ] not (a == b)
- [ ] a == (not b)
- [ ] not a == b
